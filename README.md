# Monorepo: Next.js + Firebase Functions (Gen-2)

This repo contains a Next.js To-Do app and a Firebase Cloud Functions (Gen-2) scaffold.

## Structure
- `apps/web`: Next.js app using `src/pages`
- `functions`: Firebase Functions (Gen-2) scaffold with TypeScript
- `docs`: Project documentation

## Quick start
```sh
npm install
```

## Run the web app
```sh
npm run dev
```

## Run functions emulator
```sh
npm run dev:functions
```
